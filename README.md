Repo for coding contest solutions

## How to run
There are VS code launch templates that will run the solution for the current file.

## directory structure
```
src/{contest name}/{number system}/solution.py
src/{contest name}/{number system}/input.txt
src/{contest name}/{number system}/example1.txt
src/{contest name}/{number system}/example2.txt
src/common/{stuff common to all contests}.py
```
